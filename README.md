
# Hackaton Java

En este repositorio encontraras los ejercicios resueltos de los 6 retos, ademas este readme servira para encontrar una mejor ubicación a la hora de navegar por los proyectos.

## Contenido

Cada reto esta dentro de un archivo llamado HackatonJava, asi que aquí  te brindaremos informacion sobre que reto tiene cada archivo.

1.HackatonJava: Reto El Saludador Inteligent.

2.HackatonJava2: Reto Estadistica de Notas.

3.HackatonJava3: Reto Validador de Contraseñas.

4.HackatonJava4: Reto Ordenador y Buscador.

5.HackatonJava5: Reto Sistema de Inventario Modular.

6.HackatonJava6: Reoto Calculadora con Historial Recursiva.
